### Quick start
```bash
# clone the repo
$ git clone https://github.com/yennanliu/web_development.git
# to the file
$ cd web_development
# run basic server (python 3)
$ python3 -m http.server
# open the brower and visit default : http://localhost:8000/
# click/nevigate the project sites
```
